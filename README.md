# Лабораторные работы по суперкомпьютерным вычислениям

Лабораторные работы по курсу **"Введение в суперкомпьютерные вычисления"**

**Санкт-Петербургский Государственный Университет**  
Направление: Большие данные и распределенная цифровая платформа  
3 курс, 1 семестр

## Структура репозитория

### openMP/
Лабораторные работы по технологии OpenMP для параллельного программирования с общей памятью

**Среда выполнения:** Личный ноутбук (локальная машина)

### openMPI/
Лабораторные работы по технологии MPI для параллельного программирования с распределенной памятью

**Среда выполнения:** Вычислительный кластер (SLURM)

## Технологии

- **OpenMP** - API для многопоточного программирования на системах с общей памятью
- **MPI (Message Passing Interface)** - стандарт передачи сообщений для распределенных систем
- **C/C++** - языки программирования
- **Python** - анализ данных и визуализация результатов
- **SLURM** - система управления заданиями на кластере

## Анализ результатов

Каждая задача включает:
- Исходный код на C/C++
- Скрипты для компиляции и запуска
- Скрипты Python для анализа и визуализации
- Графики производительности (время выполнения, ускорение, эффективность)
- CSV файлы с результатами измерений
